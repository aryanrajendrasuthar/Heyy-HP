# Release v0.1.0 — Sprint 1: Foundation

**Date:** 2026-04-23

---

## What shipped

Sprint 1 delivers the complete project foundation. HP launches as a real desktop
app with a window, tray icon, settings dialog, and structured logging.
No voice capability yet — that is Sprint 2.

